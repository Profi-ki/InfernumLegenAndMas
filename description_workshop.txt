This mod allows you to run the Calamity Infernum modification on Master difficulty or with the For The Worthy and Get Fixed Boi seeds on the new version.
This mod is a remaster of this mod for new version of Calamity: https://steamcommunity.com/sharedfiles/filedetails/?id=2906406399
Some scripts were copied from the Infernum modification itself : https://steamcommunity.com/sharedfiles/filedetails/?id=3459129920
